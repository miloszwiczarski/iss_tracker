Do poprawnego działania programu, należy doinstallować z poziomu desktopa plik "get-pip.py" (python get-pip.py) oraz z poziomu folderu "scripts", w folderze gdzie znajduje się zainstalowany python, "requests" (pip install requests). 
Dodatkowo opcjonalnie rozszerzenie "code runner" w visual studio codzie, aby bezpośrednio w terminalu visuala wyświetlał się na dole output.
